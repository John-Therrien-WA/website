{%- comment -%} Language switcher (EN | DE). English lives at the site root; German lives under /de/. Pages declare `lang` (en|de) and a shared `ref` key in their front matter; this finds the counterpart page by matching `ref` across languages. Pages without a `ref` (or with no translation yet) fall back to the language home page. {%- endcomment -%} {%- assign curlang = page.lang | default: 'en' -%} {%- assign en_url = '/index.html' -%} {%- assign de_url = '/de/index.html' -%} {%- if page.ref -%} {%- for p in site.pages -%} {%- if p.ref == page.ref -%} {%- if p.lang == 'de' -%}{%- assign de_url = p.url -%}{%- endif -%} {%- if p.lang == 'en' or p.lang == nil -%}{%- assign en_url = p.url -%}{%- endif -%} {%- endif -%} {%- endfor -%} {%- endif -%}
